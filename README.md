# Landing Page: GTA San Andreas

Projeto prático desenvolvido para o curso de Desenvolvimento Front-End. O objetivo do projeto é criar uma landing page temática aplicando conceitos de estruturação e estilização web.

## 🚀 Tecnologias Utilizadas
- HTML5
- CSS3 (com fontes customizadas e estilização de formulários)
- Imagens temáticas

## 🔗 Link do Projeto
O site está hospedado na Vercel e pode ser acessado no link abaixo:
projeto1-landing-page-gta-san-andreas-73r5wbnun.vercel.app

## 🛠️ Como executar o projeto localmente
1. Clone o repositório: `git clone https://github.com/VitorV9/Projeto1_LandingPage_GTASanAndreas.git`
2. Abra a pasta do projeto no seu editor de código.
3. Abra o arquivo `index.html` no seu navegador (ou use a extensão Live Server).
